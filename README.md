**Mpad Text Editor**
Mpad is a lightweight, Python-based text editor built using the tkinter library. It features a clean GUI, customizable font styles, text alignment options, and color themes to enhance your writing experience.


**🚀 Features**
Rich Text Controls: Change font families and sizes on the fly.
Styling: One-click buttons for Bold, Italic, and Underline.

Customization:
Change text color using a built-in color picker.
Choose from multiple visual themes (Light, Dark, Monokai, Night Blue, etc.).
Alignment: Support for Left, Center, and Right text justification.
Status Bar: Real-time feedback on editor status.

**🛠️ Requirements**
Python 3.x
Tkinter: Usually comes pre-installed with Python.
Icons: Ensure you have an icons2 folder in your project directory containing the .png icons referenced in the code.

**⚙️ How to Run**
Clone or download this repository.
Ensure your folder structure matches the path requirements (specifically the icons2 folder).
Run the application via terminal or command prompt:
bash
